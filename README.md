In this project, I analyzed an e-commerce customer dataset and built a Linear Regression model to predict the Yearly Amount Spent by customers. I performed data preprocessing and exploratory data analysis (EDA) to understand relationships between different variables such as session length, time spent on app/website, and membership duration.

After analyzing the data, I trained a machine learning model using regression techniques and evaluated its performance using standard metrics. I also validated model assumptions using residual analysis.
